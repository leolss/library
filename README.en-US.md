# Smart Design

## Getting Started

Install dependencies,

```bash
$ npm i
#or
$ yarn i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```

Generate a component

```bash
$ npm generate:component [name]
```
